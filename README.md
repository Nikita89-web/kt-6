# Умный загрузчик файлов

Веб-приложение для анализа текстовых файлов с использованием современных браузерных технологий.

## Описание проекта

Это учебное приложение демонстрирует работу с различными API браузера:
- Drag & Drop для загрузки файлов
- Web Workers для фоновой обработки
- Notification API для уведомлений
- WebSocket для реального времени
- Fetch API для отправки данных

## Функциональность

- Загрузка текстовых файлов (.txt) через перетаскивание или выбор
- Подсчет слов и уникальных слов в фоновом режиме
- Отправка результатов на тестовый сервер
- Системные уведомления о завершении обработки
- WebSocket соединение для обмена сообщениями
